🔵 BlueLamp AI

AI開発を加速する、ワンクリックインストール

🚀 インストール

macOS インストール

ターミナルで以下のコマンドを実行してください：

curl -O https://releases.bluelamp.ai/install.sh && chmod +x install.sh && ./install.sh

または

📥 install.sh をダウンロード

ダウンロード後、ダブルクリックで実行

Windows インストール

以下のファイルをダウンロードして、ダブルクリックで実行してください：

📥 install.bat をダウンロード

または PowerShell で：

Invoke-WebRequest -Uri "https://releases.bluelamp.ai/install.bat" -OutFile "install.bat"; .\install.bat

Linux インストール

ターミナルで以下のコマンドを実行してください：

wget https://releases.bluelamp.ai/install.sh && chmod +x install.sh && ./install.sh

Ubuntu/Debian の場合：

sudo apt update && sudo apt install python3 python3-pip curl -y

📋 システム要件

  • Python 3.8以上（3.12推奨）
  • インターネット接続
  • 100MB以上の空き容量

🎯 インストール後

インストール完了後、以下のコマンドが使用できます：

ブルーランプ拡張 # 拡張マネージャー起動
bluelamp --help # 英語版ヘルプ
